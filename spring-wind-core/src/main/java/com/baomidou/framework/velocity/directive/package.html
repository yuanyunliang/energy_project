用于在velocity中实现模板的继承，包含 block,override,extends指令
父模板：base.vm
http://badqiu.iteye.com/blog/558194
在velocity的properties中增加如下配置以激活自定义指定
userdirective=com.baomidou.velocity.directive.BlockDirective,com.baomidou.velocity.directive.OverrideDirective,com.baomidou.velocity.directive.ExtendsDirective
缓存开启
file.resource.loader.cache = true
